How it works

  1. Admin creates an estimation session
  2. Team members join using the session link
  3. Admin adds user stories and tasks
  4. Everyone votes on task estimates using planning poker cards
  5. Results are revealed when everyone votes or timer expires
  6. Admin sets the final estimate
